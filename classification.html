Projet de classification supervisée

Comparaison de cinq méthodes de classification supervisée sur différents jeux de données, avec publication sur HAL Science.

Contexte

Ce projet a été réalisé dans le cadre du Master 1 Mathématiques Appliquées à l’Université de Caen. Il consistait à étudier, présenter et implémenter cinq méthodes de classification supervisée (KNN, SVM, Arbres de décision, Random Forest et Régression Logistique) sur plusieurs jeux de données. L’objectif : évaluer et comparer leurs performances selon différents contextes.

Description du contenu

Le rapport d’environ 200 pages détaille la théorie, les implémentations en R et Python, les jeux de données utilisés, les résultats obtenus et les analyses comparatives. Il inclut également une présentation pédagogique des méthodes et du code reproductible. Le document a été publié sur HAL Science.
