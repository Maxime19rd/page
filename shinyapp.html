Application R Shiny ML

Présentation du projet

Ce projet a été réalisé dans le cadre de mon Master 2. Il s'agit d'une application R Shiny permettant de tester et comparer différentes méthodes de classification supervisée sur des jeux de données chargés par l'utilisateur.

L'utilisateur peut :

  • Sélectionner un algorithme : forêt aléatoire, SVM, k-NN, etc.
  • Ajuster les hyperparamètres
  • Visualiser les performances (accuracy, matrice de confusion, etc.)
  • Comparer plusieurs modèles de manière interactive

Technologies utilisées

  • R & R Shiny
  • Packages : shiny, caret, ggplot2, etc.
