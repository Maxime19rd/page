Application R Shiny ML

Présentation du projet

Ce projet a été réalisé dans le cadre de mon Master 2. Il s'agit d'une application R Shiny permettant de tester et comparer différentes méthodes de classification supervisée sur des jeux de données chargés par l'utilisateur.

L'utilisateur peut sélectionner un algorithme (forêt aléatoire, SVM, k-NN, etc.), ajuster les hyperparamètres, visualiser les performances (accuracy, matrice de confusion, etc.) et comparer les résultats. L’objectif est de rendre ces méthodes accessibles et interactives à travers une interface intuitive.
